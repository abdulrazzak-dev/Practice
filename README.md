# # Java Programming - Practical Tasks

This repository contains my solutions for the Java practical task modules assigned by MARS TECH. Every program is written in Java, utilizes the `Scanner` class for dynamic user input, and follows proper coding and documentation standards.

## 📁 Repository Structure

The project is organized into logical topic folders under the main task directory:

* **`01. Java Operators and Control Flow Statements/`** — Basics of syntax, arithmetic operations, conditional logic (`if-else`, `else-if`), and `switch` statements.
* **`02. Java While Loop/`** — Iterative logic tasks ranging from basic counters to mathematical algorithms and data manipulation using `while` loops.
* **`03. Java For Loop & Nested For Loop/`** — Pattern printing, math sequences, and coordinate-like multi-line structural outputs using nested loop variations.
